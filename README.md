## 🧾 Snapshot
Mình là **Toàn**. Mình thích những công việc cần **tư duy hệ thống**: nhìn tổng thể, đọc tín hiệu (traffic/logs), tìm nguyên nhân gốc, và biến mọi thứ thành quy trình/ghi chép có thể tái sử dụng.

Nếu mô tả ngắn gọn phong cách làm việc của mình:
- **Build** (web/tools) → để “đưa ý tưởng ra sản phẩm”
- **Troubleshoot** (network/system) → để “xử lý vấn đề nhanh & đúng”
- **Observe** (logs/packets) → để “đọc được chuyện gì đang xảy ra”
- **Automate + Document** → để “làm việc có hệ thống, không lặp sai”
